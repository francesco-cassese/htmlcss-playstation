# Progetto: Replica Layout Home Page PlayStation

Esercizio di ricostruzione della home page di PlayStation basato su riferimenti statici (screenshot), con l'obiettivo di padroneggiare il **Grid System di Bootstrap 5**.

## 🎯 Obiettivo del Progetto
L'obiettivo principale è testare la capacità di analisi di un layout reale e la sua traduzione in codice utilizzando esclusivamente (o quasi) le classi di Bootstrap 5.

### Focus Tecnico:
* **Grid System & Flexbox:** Implementazione di una struttura flessibile che si adatta ai vari viewport (Mobile, Tablet, Desktop).
* **Sincronizzazione Breakpoint:** Gestione fluida del passaggio da layout a colonna singola a strutture multi-colonna.
* **Griglie Annidate:** Utilizzo strategico di `.row` all'interno di `.col` per replicare sezioni complesse come le griglie dei prodotti o delle news.
* **Utility Classes:** Ampio utilizzo di classi per margini (`m-*`), padding (`p-*`), display e posizionamento.

## 🛠️ Tecnologie Utilizzate
* **HTML5:** Strutturazione semantica dei contenuti.
* **CSS3:** Personalizzazioni specifiche dove le utility di Bootstrap non sono sufficienti.
* **Bootstrap 5.3:** Framework principale per il layout responsive e componenti (Card, Navbar, Button).
